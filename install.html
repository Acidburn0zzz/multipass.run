Get started with Multipass

Installing Multipass is quick and easy. Follow the instructions for the Operating System of your choice and multipass will be available on your command line.

Installing Multipass

Linux

Requirements:

  • OS X 10.10.3 Yosemite or later
  • a 2010 or later Mac (i.e. a CPU that supports EPT)

Download the installer, run it and follow the on-screen instructions.

Then, start the Terminal app and run multipass help to see the available commands.

Next steps

Once you have installed Multipass, run multipass find to list the images available, multipass launch to just launch the default one or multipass help to see all the available commands.

Multipass is integrated into many workflows and tools that were normally limited only to Linux. Now you can work with Ubuntu without having to adapt your work environment.
