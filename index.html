Instant Ubuntu VMs

For cloud and Kubernetes developers, turn your workstation into a free mini-cloud with a CLI to launch instances and drive cloud-init.

Features

Develop for Ubuntu

Ubuntu is the leading operating system in the cloud and IoT. With Multipass you can target it directly from your developer OS of choice.

Always up to date

Multipass always uses the newest and most up-to-date image available, and will maintain those you use often.

Easy to use

Getting a new instance is as easy as multipass launch.

Get started

Multipass is available for macOS and Windows, and the many Linux distributions that support snaps.

See all the distributions that run snapd

Once you've installed Multipass, check out multipass find to see what images are available for you and start one via multipass launch. See multipass help for information on all the commands.
